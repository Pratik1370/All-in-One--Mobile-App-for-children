1.0 Introduction to App

All in one is an ultimate web app proposed as a learning tool for kids. It is an educational app which aims to aid basic learning features among kids. It is targeted for the preschool and kindergarten children. With the use of flash cards, numerous arenas are covered which ultimately help kids to easily grab different things in a visual and audio manner. Further, the testing and practice tool could make kids more interested in challenges and develop a feeling of accomplishing them. Basic features that are targeted to include in the application comprises of flashcards for alphabets, numbers, months, days, counting, vegetables, fruits, vehicles, shapes, colours and basic maths learning. Practice tool comprises of several games as memory test, counting test, quiz, and the interactive colour book aims to make the app more interesting for kids to learn colouring through digital platform. In nutshell, it is a complete package that would be a helpful tool for making children learn in an interactive medium.

Application name: All in one: Ultimate learning tool for kids

Targeted group: Kids who are in preschool or kindergarten

2.0 Background of an application

The current world is slowly moving towards the era of information technology. The practice of use of technology in teaching and learning is currently at peak and flourishing. Several researches even prove that the use of technology could be the best medium for children to learn and grab different information in no time.
According to (Campbell et al. 2006) , “for the most part, children learned to use the targeted device and gained competence in device use through practice.” The review paper clearly demonstrates how easily children grab the information which is imparted using technology. Further (Bates, W. & Poole 2003) also demonstrates the use of technology in higher education which clearly indicates how efficient is technology in field of teaching learning process.
Keeping these concepts in concern, the application “All in one: Ultimate learning tool for kids” is coined.
